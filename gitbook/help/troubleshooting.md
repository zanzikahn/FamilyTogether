## Data Problems

<details>
<summary><strong>ğŸ’¨ All my data is gone!</strong></summary>

### Possible causes:
- Browser data was cleared
- Different browser/device
- Someone clicked "Clear All"
- Private browsing mode

### Solutions:

{% hint style="success" %}
**If you have a backup file:**
1. Open the app
2. Go to Admin Panel
3. Click "Import Backup"
4. Select your backup file
{% endhint %}

{% hint style="danger" %}
**If you don't have a backup:**
Unfortunately, the data cannot be recovered. You'll need to start fresh with "Clear All" and re-create everything.

**Lesson learned:** Always keep backup files!
{% endhint %}

</details>

<details>
<summary><strong>ğŸ“¤ Export/Import isn't working</strong></summary>

### For Export problems:
- Check if downloads are blocked in your browser
- Try a different browser
- Check your Downloads folder

### For Import problems:
- Make sure the file is a `.json` file
- Make sure it's a FamilyTogether backup (not some other JSON)
- File may be corrupted â€” try a different backup

</details>
