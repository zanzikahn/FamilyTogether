# Contact Support

Need help that isn't covered in the documentation? Here's how to get support.

## Who to Ask

{% hint style="info" %}
**Daddy is your FamilyTogether support person!**

He created this app and knows how everything works.
{% endhint %}

---

## Before Asking for Help

Try these steps first:

{% stepper %}
{% step %}
### Check the Documentation

Your answer might already be here!

- [Troubleshooting](troubleshooting.md) ‚Äî Common problems and solutions
- [FAQ](faq.md) ‚Äî Frequently asked questions
{% endstep %}

{% step %}
### Try Basic Fixes

Many issues can be fixed by:
- Refreshing the page (F5)
- Logging out and back in
- Trying a different browser
- Waiting a few minutes (for lockouts)
{% endstep %}

{% step %}
### Gather Information

If you still need help, collect this info:
- What were you trying to do?
- What happened instead?
- Any error messages?
- What browser are you using?
{% endstep %}
{% endstepper %}

---

## How to Report a Problem

### For Kids

1. **Tell Daddy** what's happening
2. **Show him the screen** if possible
3. **Explain what you were trying to do**

### For Parents

1. **Note the exact issue** ‚Äî what happened?
2. **Screenshot the problem** if possible
3. **Check if it's reproducible** ‚Äî does it happen every time?
4. **Try the troubleshooting steps** first

---

## Information to Include

When reporting a problem, these details help:

| Information | Example |
|-------------|---------|
| What happened? | "I clicked submit but nothing happened" |
| What did you expect? | "The task should turn yellow" |
| Browser | Chrome, Firefox, Safari, Edge |
| Device | Computer, tablet, phone |
| When did it start? | "Today" or "After the last update" |
| Error message? | Screenshot or exact text |
