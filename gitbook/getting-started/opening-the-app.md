# Opening the App

FamilyTogether is a web-based app that runs directly in your browser. No installation required!

## How to Open FamilyTogether

{% tabs %}
{% tab title="On a Computer" %}
### Windows / Mac / Linux

1. Navigate to the FamilyTogether folder on your computer
2. Find the file named `index.html`
3. Double-click the file to open it in your default web browser
4. The login screen will appear

**Folder Location:**
```
C:\Users\Zanzi\TOOLS\FamilyTogether\Version_2_Complete\index.html
```
{% endtab %}

{% tab title="On a Tablet/Phone" %}
### Mobile Devices

1. Open your web browser (Safari, Chrome, etc.)
2. Navigate to where the file is hosted or stored
3. The app will automatically adjust to fit your screen

{% hint style="info" %}
Ask Daddy to help set up a bookmark or home screen shortcut for easy access!
{% endhint %}
{% endtab %}
{% endtabs %}

## Creating a Shortcut (Optional)

{% hint style="success" %}
**Pro Tip:** Create a desktop shortcut for quick access!
{% endhint %}

{% tabs %}
{% tab title="Windows" %}
1. Right-click on `index.html`
2. Select "Create shortcut"
3. Drag the shortcut to your Desktop
4. Rename it to "FamilyTogether"
{% endtab %}

{% tab title="Mac" %}
1. Open the file in Safari or Chrome
2. Drag the URL from the address bar to your Desktop
3. Rename the bookmark to "FamilyTogether"
{% endtab %}
{% endtabs %}
