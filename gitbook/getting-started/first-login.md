# Your First Login

Welcome! This page will help you log in to FamilyTogether for the first time.

## Logging In

{% stepper %}
{% step %}
### Find Your Name

Look at the login screen and find the button with your name and avatar (emoji picture).

Each button shows:
- Your avatar emoji
- Your name
- Your current points
- "Parent" badge (if you're a parent)
{% endstep %}

{% step %}
### Click Your Profile

Tap or click on your name. The screen will change to show a password box.
{% endstep %}

{% step %}
### Enter Your Password

Type your password into the box. 

{% hint style="info" %}
Can't remember your password? Ask Daddy to tell you or reset it for you!
{% endhint %}
{% endstep %}

{% step %}
### Click "Login"

Press the blue **üîë Login** button.

If your password is correct, you'll see your dashboard!
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
**Keep Your Password Secret!**

If someone else uses your account, they could:
- Complete tasks as you
- Spend your hard-earned points
- See your personal information
{% endhint %}

## What If I Enter the Wrong Password?

Don't worry! Everyone makes mistakes.

| Attempt | What Happens |
|---------|--------------|
| 1st wrong try | Warning message shows |
| 2nd wrong try | Warning shows attempts left |
| 3rd wrong try | **Account locked for 5 minutes** |

{% hint style="danger" %}
**Account Locked?**

If you see "Account locked for 5 minutes":
1. Wait 5 minutes
2. Make sure you know the correct password
3. Try again carefully
{% endhint %}

## Logging Out

When you're done, always log out:

1. Look for the **Logout** button in the top-right corner
2. Click it to return to the login screen
3. Your progress is automatically saved!

{% hint style="success" %}
**Good Habit:** Always log out when you're finished, especially on shared devices!
{% endhint %}

---

## Switching Users

If someone else wants to use the app:

1. Click **Logout** 
2. The login screen will appear
3. The other person can select their profile and log in
