# Welcome to FamilyTogether

<figure><img src=".gitbook/assets/logo-placeholder.png" alt="FamilyTogether Logo"><figcaption><p>FamilyTogether - Family Cleaning Tracker</p></figcaption></figure>

## What is FamilyTogether?

FamilyTogether is a fun, easy-to-use app that helps your whole family work together to keep your home clean and tidy. It turns everyday chores into an exciting game where everyone can earn points and win rewards!

Think of it like a video game for real life: complete tasks, earn points, and trade those points for special treats like extra screen time, choosing dinner, or having a friend sleep over.

{% hint style="info" %}
**Works Anywhere!**

FamilyTogether runs in any web browser on your computer, tablet, or phone. No app download needed!
{% endhint %}

## Quick Start Guide

{% stepper %}
{% step %}
### Open FamilyTogether

Double-click the `index.html` file or open it in your web browser.
{% endstep %}

{% step %}
### Select Your Profile

Click on your name from the list of family members.
{% endstep %}

{% step %}
### Enter Your Password

Type your password and click "Login".
{% endstep %}

{% step %}
### Start Completing Tasks!

Find tasks assigned to you and click "Submit for Approval" when done.
{% endstep %}
{% endstepper %}
