## Individual Task Setting

{% hint style="info" %}
**Check "Individual Task" when EACH person needs to do it separately.**
{% endhint %}

### When to Check â˜‘ï¸ Individual Task:

| Task | Individual? | Why |
|------|-------------|-----|
| Make beds | â˜‘ï¸ Yes | Each person makes their OWN bed |
| Put away own clothes | â˜‘ï¸ Yes | Each person handles their own |
| Vacuum living room | âŒ No | Only needs to be done once |
| Wash dishes | âŒ No | One person does it |

### How It Works:

**Individual = ON:**
- Each assigned person sees the task
- Each person must complete it separately
- Completion shows "2/4 done" style counter
- Each person earns points when they complete their part

**Individual = OFF:**
- Only one person needs to complete it
- First person to complete gets the points
- Shows as done for everyone after completion

## Setting Point Values

Use this guide to set fair point values:

| Task Difficulty | Time | Suggested Points |
|-----------------|------|------------------|
| Very Easy | 2-5 min | 1-2 points |
| Easy | 5-10 min | 2-4 points |
| Medium | 10-20 min | 4-8 points |
| Hard | 20-45 min | 8-15 points |
| Very Hard | 45+ min | 15-25 points |

{% hint style="success" %}
**Tip:** Harder or less popular tasks should be worth more points to motivate completion!
{% endhint %}
