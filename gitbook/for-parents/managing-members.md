## Resetting a Password

If someone forgets their password:

1. Go to Admin Panel
2. Find them in Family Members
3. Click **"âœï¸ Edit"**
4. Look at or change the password field
5. Tell them the password
6. Click **"ğŸ’¾ Save"** if you changed it

{% hint style="info" %}
**Passwords are visible to parents in the edit screen.**

This makes it easy to help kids who forget their passwords!
{% endhint %}

## Adjusting Points Manually

Sometimes you need to adjust someone's points:

**Reasons to ADD points:**
- Completed a task outside the app
- Bonus for exceptional work
- Making up for a mistake
- Birthday bonus!

**Reasons to REMOVE points:**
- Correcting an error
- Consequence for behavior
- Points given by mistake

### How to Adjust:

1. Click **"âœï¸ Edit"** on their profile
2. Change the Points value
3. Click **"ğŸ’¾ Save"**

{% hint style="warning" %}
**Be Fair!**

Manual point adjustments should be rare and justified. Kids should primarily earn points through task completion.
{% endhint %}

## Password Best Practices

{% tabs %}
{% tab title="For Young Kids" %}
### Simple & Memorable

- Use short, easy words
- Something they can remember
- Example: `puppy1` `star123` `happy`

{% hint style="success" %}
**Tip:** Let them choose their own simple password â€” they'll remember it better!
{% endhint %}
{% endtab %}

{% tab title="For Older Kids/Teens" %}
### More Secure

- Longer passwords
- Mix of letters and numbers
- Something meaningful to them
- Example: `soccer2024` `gaming4life`
{% endtab %}

{% tab title="For Parents" %}
### Secure

- Don't use easy-to-guess passwords
- Different from other accounts
- Remember: kids might try to guess it!
{% endtab %}
{% endtabs %}
