## The Reward Store

Below your points, you'll see the **Reward Store** with cards for each available reward.

### Understanding Reward Cards

{% tabs %}
{% tab title="Can Afford âœ…" %}
### Purple Border = You Can Buy It!

When a reward has a **purple border** and purple background:
- You have enough points
- Click to claim it instantly!
- Points will be deducted
{% endtab %}

{% tab title="Can't Afford Yet âŒ" %}
### Gray Border = Need More Points

When a reward has a **gray border**:
- You don't have enough points yet
- The card shows how many more you need
- Keep completing tasks!
{% endtab %}
{% endtabs %}

### Reward Card Information

Each reward card shows:
- **Name:** What you'll get (e.g., "30 min extra screen time")
- **Category:** Type of reward (Screen Time, Treats, etc.)
- **Cost:** How many points needed (purple badge)

## Individual vs. Group Tasks

{% hint style="info" %}
**Look for the "Individual task" badge!**

Some tasks marked "Everyone" need to be done by each person separately.
{% endhint %}

### Examples:

| Task | Type | Meaning |
|------|------|---------|
| "Make beds" | Individual | Each person makes their OWN bed |
| "Vacuum living room" | Group | Only ONE person needs to do it |

For individual tasks, the "Completed" counter shows how many family members have done it (e.g., "2/4").
